관리자 설정

OpenRouter 모델과 API 키를 설정하고 연결을 테스트합니다

1. 모델 · API 키

여기서 입력한 값은 이 페이지 안에서 연결 테스트를 하거나 .env 파일을 만드는 데만 사용됩니다. 이 페이지 자체는 저장 기능이 없습니다.

비용 발생을 막기 위해 유료 모델은 선택할 수 없도록 제한되어 있습니다.

⚠️ :free 모델은 계정 누적 충전액이 10크레딧 미만이면 하루 50회, 10크레딧 이상이면 하루 1,000회로 요청이 제한됩니다(분당 20회는 공통). 20~50명이 한 번에 응시하면 하루 120~300회가 필요하므로, 무료 모델을 쓰려면 먼저 10크레딧 이상 충전해 두는 것이 안전합니다.

키는 openrouter.ai/settings/keys 에서 발급받습니다.

저장하면 이 컴퓨터의 이 브라우저에서 settings.html을 다시 열 때 모델·키가 자동으로 채워집니다. (다른 컴퓨터나 다른 사람에게는 전달되지 않습니다.)

3. 설정 파일(.env) 적용

아래 내용을 이 폴더의 .env 파일에 그대로 덮어쓰면, 학생들은 이름·학번만 입력하고 바로 평가를 시작할 수 있습니다.

  1. 아래 .env 다운로드 버튼을 누릅니다.
  2. 다운로드된 .env 파일을, index.html과 같은 폴더에 있는 기존 .env 위에 덮어씁니다.
  3. 학생에게 index.html과 .env 두 파일을 함께 전달합니다 (압축해서 보내면 편합니다).

.env를 바꾼 뒤에는 index.html을 새로고침해야 반영됩니다.

⚠️ 브라우저가 저장 대화상자에서 파일 이름을 .env가 아닌 다른 이름(예: env.js)으로 자동으로 바꿔 제안할 수 있습니다. 그런 경우 저장할 때 파일 이름 칸에 직접 .env라고 입력하거나, 저장 후 파일 탐색기에서 이름 바꾸기(F2)로 .env로 고쳐주세요. 확장자를 바꾸겠냐는 경고가 뜨면 "예"를 누르시면 됩니다.
